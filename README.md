coreJS Selenium Workshop
=================

A simple Selenium project example


1. Install dependencies
  * [Node.js](http://nodejs.org/)
  * [Java](http://www.oracle.com/technetwork/java/javase/downloads/index.html)
2. Init project
  ```bash
  npm install
  ```
3. Run it!
  ```bash
  grunt test:selenium
  ```
